# Renormalized-Angular-Momentum-Expansions---Bachelor-Thesis-Simon-Schmitt


The following code is based on and accompanies the Bachelor thesis of Simon Schmitt titled "Connecting Monodromies and The Renormalized Angular Momentum in Kerr-de Sitter Spacetimes". The code provides expansions for the renormalized angular momentum parameter, introduced in the MST formalism, in the the rotating Nariai limit in terms of the extremality parameter \eta=(r_C-r_+)/L.

The code is written in Wolfram Mathematica and provided in notebook form. 

It should be noted that the code generates to files, in which the central object, the continued fraction equation is stored after a first execution to save compute time.
